# poke_api

Use of AI
Artificial intelligence was used as a learning and debugging resource throughout the development of this project. AI assistance was used to explain programming concepts, help troubleshoot errors, clarify API documentation, and provide guidance on structuring routes and game logic. No code was copied directly into the project without review. Instead, suggestions were used as a reference to support learning and problem-solving, and all code was written, tested, modified, and integrated by the developer. AI served as a tool to enhance understanding of the technologies used rather than to replace the software development process.
